# 🚀 Лендинг для рекламы мастеров

Яркий и современный лендинг для продвижения услуг рекламы. По умолчанию настроен для мастеров пластиковых окон, но легко адаптируется под любую нишу через JSON.

## ✨ Особенности

- 🎨 **Яркий дизайн** с градиентами и анимациями
- 📱 **Адаптивный** для всех устройств
- 🔄 **Гибкая настройка** через JSON файл
- ⚡ **Быстрый** на Vite
- 🎯 **Конверсионные кнопки** на каждом экране
- 💬 **Блок отзывов** с рейтингами
- 💳 **Рассрочка** и бесплатный месяц
- ✅ **Модальное окно** подтверждения отправки

## 🚀 Быстрый старт

1. **Установите зависимости:**
```bash
npm install
```

2. **Запустите dev-сервер:**
```bash
npm run dev
```

3. **Откройте в браузере:**
```
http://localhost:5173
```

## 📝 Настройка под свою нишу

Отредактируйте файл `src/data.json`:

```json
{
  "niche": "Ваша ниша",
  "title": "Заголовок сайта",
  "hero": {
    "heading": "Главный заголовок",
    "subheading": "Подзаголовок",
    "cta": "Текст кнопки"
  },
  "features": [...],
  "reviews": [...],
  "installment": {...},
  "freeTrial": {...}
}
```

### Примеры ниш:
- Мастера по ремонту
- Стоматологии
- Автосервисы
- Салоны красоты
- Юридические услуги
- И любые другие!

## 🛠 Команды

- `npm run dev` - Запуск dev-сервера
- `npm run build` - Сборка для продакшена
- `npm run preview` - Просмотр production сборки

## 📦 Билд для продакшена

```bash
npm run build
```

Готовые файлы будут в папке `dist/`

## 🎯 Основные блоки

1. **Hero** - Главный экран с заголовком и кнопкой
2. **Features** - Преимущества (4 карточки)
3. **Free Trial** - Бесплатный месяц
4. **Reviews** - Отзывы клиентов (3 карточки)
5. **Installment** - Рассрочка и цены
6. **CTA Form** - Форма заявки
7. **Footer** - Контакты

## 💡 Фишки

- Все кнопки ведут к форме заявки
- Плавная прокрутка
- Анимации появления элементов
- Hover-эффекты на карточках
- Адаптивная типографика

## 📞 Контакты

Измените контакты в файле `src/data.json` в секции `cta`:
```json
"cta": {
  "phone": "+7 (999) 123-45-67",
  "email": "info@example.com"
}
```

## 🎉 Готово!

Ваш лендинг готов к использованию. Измените данные в JSON и получите лендинг под вашу нишу!
